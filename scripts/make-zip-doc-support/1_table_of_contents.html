Provider Screening Module documentation

This bundle contains several user, administrator, and developer guides for the Provider Screening Module.

Here's a guide to the files. Please extract this bundle onto your computer to make the links work.

Use

The user manual for providers, service agents, state Medicaid staff, and system administrators, 2_usermanual.pdf, covers:

  • Creating and submitting enrollments
  • Verifying, approving and rejecting enrollments
  • Creating an account and logging in
  • Managing users

Understand

Read the general guide to the PSM's codebase (3_readme.pdf) and the guide to its technical architecture and design (4_architecture-design.pdf), and view the PSM architectural diagram (5_architecture-for-stakeholders-diagram.pdf) and PSM Interface Control Document (6_interface_control_document.pdf).

View diagrams describing the enrollment process (7_enrollment-process-diagram.png) and the screening process (8_screening-process-diagram.png).

View a compact diagram of the PSM's database schema (9_schema-compact-diagram.png) and a hierarchically organized diagram of the PSM's database schema (10_schema-hierarchical-diagram.png).

Install and Run

Learn to install the application (11_install-instructions.pdf), and understand its external technical dependencies (12_dependencies-list.pdf).

Learn to extract, transform, and load (ETL) verification data from the List of Excluded Individuals/Entities (LEIE) on the command line (13_leie_README.pdf) for use in your PSM instance, and understand how the LEIE formats its data (14_leie_API.pdf).

Develop

Learn to test (15_testing-instructions.pdf) and contribute to the PSM as a developer (16_developer-contribution-instructions.pdf), or use its web API (api-docs/index.html).
